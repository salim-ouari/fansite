FIFA ULTIMATE TEAM CHAMPIONS


FUT Champions est le summum du contenu compétitif pour la plupart des joueurs de FIFA 22 Ultimate Team. L'idée est de jouer 20 matchs entre le vendredi et le lundi matin, d'accumuler un tas de victoires et de profiter de récompenses uniques.
C'est un moyen de tester votre niveau lors d’une compétition en deux phases : les Playoffs et les Finales. Jouez pour gagner de superbes récompenses et montrez votre talent sur le terrain.

PLAY-OFF FUT CHAMPIONS

Les Playoffs représentent la première phase de FUT Champions. Ils sont ouverts pendant toute la durée de la Saison FUT afin que vous puissiez les disputer à votre rythme. Il existe un nombre limité de participations aux Playoffs par Saison FUT, comme vous pourrez le voir dans le jeu. Cette limite est réinitialisée au début de chaque nouvelle Saison, qui se produit généralement toutes les 6 semaines.

Comment se qualifier pour les Playoffs ?

Jouez dans Division Rivals pour gagner des Points de qualification Champions (PQC). Quand vous aurez atteint le nombre requis de PQC pour participer aux Playoffs, vous pourrez disputer automatiquement la compétition en cours. Vous pouvez vérifier dans le jeu combien de PQC sont nécessaires pour vous qualifier pour les Playoffs.

FINALES FUT CHAMPIONS

Les Finales représentent la deuxième phase de FUT Champions. Consultez le dernier calendrier dans le jeu car les Finales ne sont accessibles qu’à certaines périodes.

Comment se qualifier pour les Finales ?

Vous pourrez vous qualifier pour les Finales FUT Champions en remportant suffisamment de points dans les Playoffs FUT Champions pour obtenir un jeton Qualification Finales. Vous choisirez alors quand vous souhaitez participer aux Finales en utilisant le jeton.
